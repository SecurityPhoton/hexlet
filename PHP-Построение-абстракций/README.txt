Анонимные функции - функции без имени. Анонимные функции допускается
 передавать в качестве параметров другим функциям.

Замыкание - функция, которая запоминает состояние окружения в момент своего создания.
 Даже если состояние затем измениться, замыкание содержит в себе первоначальное состояние.
 Замыкание применимо только к анонимным функциям. Для активизации необходимо использовать ключевое слово use.
 После Use можно указать переменные которые должны войти в замыкание.
 Основное назначение замыканий - замена глобальных переменных. В отличие от глобальных переменных, вы можете передать
 внутрь функции значение но уже не сможете изменить переменую переданую через механизм замыкания.
